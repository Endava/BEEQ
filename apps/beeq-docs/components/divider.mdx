## Usage guidelines

In a design system, dividers serve to visually separate content and provide a clear hierarchy.

### When to use

Use dividers when you want to create a visual distinction between content sections, promoting a clearer and more organized layout. They are particularly useful for distinguishing different elements within a user interface.

### When not to use

Dividers should be avoided in scenarios where visual separation is unnecessary or can lead to clutter.

Dividers primarily serve as decorative elements and are not subject to specific accessibility requirements.

Dividers should be excluded from screen reader accessibility considerations.

## Properties/attributes

| Property | Attribute | Description | Type | Default |
| --- | --- | --- | --- | --- |
| **dashed** | **dashed** | If true, the divider has a dashed pattern | `boolean` | `false` |
| **orientation** | **orientation** | The default orientation of the divider | `horizontal` `vertical` | `horizontal` |
| **strokeBasis** | **stroke-basis** | Set the min width of the divider's stroke when the text is not centered. (*the value expressed in px)* | `number` | `0` |
| **strokeColor** | **stroke-color** | Set the stroke color of the divider *(the value should be a valid value of the palette color).* | `string` | `stroke--primary` |
| **strokeDashGap** | **stroke-dash-gap** | Set the gap of the divider's stroke. This is applicable when the stroke is dashed | `number` | `7` |
| **strokeDashWidth** | **stroke-dash-width** | Set the width of each dash of the divider's stroke. This is applicable when the stroke is dashed | `number` | `12` |
| **strokeLinecap** | **stroke-linecap** | Set the line of the divider's stroke. This is applicable when the stroke is dashed | `butt` `round` `square` | `butt` |
| **strokeThickness** | **stroke-thickness** | Set the thickness of the divider's stroke *(the value expressed in px)*. | `number` | `1` |
| **titleAlignment** | **title-alignment** | Set the alignment of the title on the main axis of the divider (horizontal/vertical) | `end` `middle` `start` | `middle` |
