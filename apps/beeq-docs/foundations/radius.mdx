Radius helps emphasize and highlight specific design features. Rounded corners feel more friendly, and add extra perceived fidelity to the visual elements. Larger components that contain other components can have a greater corner radius. Each component comes with the appropriate corner radius already applied.

| **Corner radius value** | **Used for** |
| --- | --- |
| `radius-full` 360px | Used primarily to create a circle and often as a background to an icon or other graphic elements. |
| `radius-xl` 32px | Extra-large components, cards. |
| `radius-l` 24px | Large container components. |
| `radius-m` 12px | Medium components like cards, snack bars, and banners. |
| `radius-s` 8px | Medium or small components that often live inside others, like buttons or nested components. |
| `radius-xs` 4px | Small components. |
| `radius-xs2` 2px (icons) | Used primarily on small shapes to maintain a balance between the shape and the border-radius. |

## Nested components

<CardGroup cols={1}>
  <Card title="Interior components have a smaller corner radius">
    When working with nested elements to achieve an aesthetic appearance, adjusting border-radius values is essential. The crucial factor influencing the appearance is the gap between the two elements. To achieve an appealing look, ensure that the corner radius of the outer element is greater than that of the interior components. A helpful formula is to subtract the gap between the elements from the border-radius value of the outer element, resulting in a seamless and polished design.
