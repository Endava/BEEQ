By using shadows thoughtfully and consistently, we can create a more immersive and engaging experience for our users. Shadows help distinguish between interactive and static elements, indicate elevation or layering, and draw attention to specific components when needed.

---

## The importance of elevation

Shadows can improve overall aesthetic, add depth and realism to the user's visual experience, and improve the visual hierarchy. This helps users discover and interact with UI elements.

Using elevation in design systems has several benefits, including:

- **Creating a visual hierarchy:** Elevations can be used to differentiate between different elements and prioritize them based on their importance or hierarchy.
- **Providing depth and dimensionality:** Elevation can create the illusion of three-dimensionality, making elements appear more interactive and engaging.
- **Guiding focus:** Elevation can be used to guide users' focus through layering or indicate interactive UI elements.
- **Enhancing scannability:** Elevation can help users quickly scan and identify important elements within a design system.
- **Indicating interactivity:** Elevation can communicate that an element can be interacted with, providing users with visual feedback about the availability and functionality of different UI components.
