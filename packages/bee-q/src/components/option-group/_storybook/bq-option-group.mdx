<Title>Option Groups</Title>

The Option Group component is a visual and functional grouping mechanism for organizing related options in a hierarchical manner.
It typically consists of a label or title that represents the group's category and a list of associated options nested under the label. When the dropdown is expanded, the options within the group are displayed as part of the larger list of options, making the hierarchy clear to the users.

<Subtitle>Usage</Subtitle>

Typically it is used whitin a dropdown to display a list of option. It has the header of the group containing 3 different slots (`header-prefix`, `header-label`, `header-suffix`)
and the slot element for the list of options.

It is used with the `<bq-dropdown>` component and `<bq-option-list>` component, typically to display a dropdown list.

<Subtitle>üëç When to use</Subtitle>

Use it when you want to provide a way to categorize and present a large set of options more intuitively, making it easier for users to find and select the desired option.

<Subtitle>Benefits</Subtitle>

- Improved organization and UX
- Enhanced user experience
- Better accessibility
- Consistency and scalability (promotes a unified look and feel)
