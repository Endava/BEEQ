    <Title>Table</Title>

    The Table component is a UI element that provides a table-like interface to display data in a structured format.
    The table is not a custom element, actually BEEQ only provides the styles rules for the native HTML table element.

    <Subtitle>Usage</Subtitle>

    Here are some best practices to follow when using native HTML table element:

    - Use the `bq-table` class on the table element.
    - For responsive tables, wrap the table in a `bq-table--container` element.
    - For compact tables, add the `compact` CSS class to the table element.
    - For bordered tables, add the `bordered` CSS class to the table element.
    - For making a row selected, add the `selected` CSS class to the row element.

    <Subtitle>üëç When to use</Subtitle>

    The HTML table element is ideal for use in the following scenarios:

    - When you need to display data in a clear and organized manner.
